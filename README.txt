INTRODUCTION
------------
The whole project contains four files which are README.txt, tokenizer.cpp, tokenizer.h and main.cpp.
tokenizer.h and tokenizer.cpp define a class used to read input file and transfer to tokens.
main.cpp contains a bunch of methods and global vairabls to address error input and output symbol table, warning and error message

REQUIREMENTS
------------
To run this application requires the compiler support at least c++11

INSTRUCTION
------------
To run this application please type commands below into command line:
module load gcc-6.2.0
g++ -o test main.cpp tokenizer.cpp
./test input.txt
